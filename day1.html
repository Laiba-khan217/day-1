"Introduction to HTML"

HTML(Hyper Text Markup Language) is the standard language used to create the structure web pages. it defines the basic layout of a webpage using elements like headings, paragraphs, links, images, and tables. HTML uses tags enclosedv in angle brackets to tell the browser how to display content. While it does not handle styling or interactivity by itself, HTML works together with CSS for design and JavaScript for functionality, forming the foundation of modern web development.
